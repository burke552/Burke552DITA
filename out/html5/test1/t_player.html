Creating a player character for a video game in <canvas>

This document instructs on how write a player character or an enemy character in a video game.

To accomplish this goal you will need to:

  • Set up an object containing variables of the player character and modify them.
  • Choose to make the player character after setting up a character.
  1. Enable user input by writing keys = [].
  2. Modify velocity variables in response to player input inside the update loop.
    1. Write if statements for the keys that correspond to up, left and right: if (keys[X]){
    2. For the "up" if statement, set player.isJumping to true, set player.speed as negative and assign that value to player.yVelocity.
    3. For the "left" if statement, write an if statement that adds 1 to player.xVelocity if it is less than player.speed.
    4. For the "right" if statement, write an if statement that subtracts 1 from player.xVelocity if it is greater than player.speed.
  3. Update the position variables based on the velocity variables in the update loop.
    1. Set player.xVelocity equal to its current value multiplied by the friction variable.
    2. Set player.yVelocity equal to its current value plus the gravity variable.
    3. Set player.x and player.y equal to player.xVelocity and player.yVelocity.
    4. Write a series of if statements checking if player.x is less than 0 or greater than the width of the canvas.
    5. Set player.x equal to the width of the canvas if it exceeds that value, or set player.x to 0 if it is less than 0..
    6. Write an if statement checking if player.y is less than 0.
    7. Set player.isJumping to false and player.y to 0 if it is less than 0.
